
WinnersTour — Topbar Patch
==========================
1️⃣ Copie a pasta `assets/` inteira para seu projeto (mantendo `assets/css/patches/`).
2️⃣ No `<head>` do index.html, logo após o base.css:
   <link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;600;700;900&display=swap" rel="stylesheet">
   <link rel="stylesheet" href="assets/css/patches/wt-topbar.css">
3️⃣ Substitua o header antigo pelo novo snippet (menu à esquerda, título central, WhatsApp à direita).
